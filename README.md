# Redux

Q. Why Redux?
A. 리액트의 useState만으로는 state의 component-level접근/변경만 가능하다는 단점에서 벗어나, 그 외 외부 컴포넌트에서 state에 접근/관리하기 위해 리덕스를 사용합니다.
